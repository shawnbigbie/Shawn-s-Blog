Final Project

Posted by Shawn Bigbie on Nov 7th, 2020

What have you done this week?

During this week (Nov 1st - Nov 8th) I have started seting up my final project for DGM 3760 which is called My Library. This project is useing Express, Node Js, MongoDB for the backend, and EJS for the template engine. The point of this project is for users to store and create, edit, and delete authors and books from a public database.

Below are the things I have created and worked on.

  • Setup Express and Dependencies
  • Create database MongoDB Atlas
  • Hooked up MongoDB
  • Setup Github Repo
  • Setup .env
  • Create book and author model
  • Create book and author routes
  • Create book, author, layout, and other views needed
  • Setup a shared page header
  • Setup error handling

What do you plan on doing next week?

Things that I still need to work on or finish.

  • Create Public Folder
  • Create FrontEnd CSS & HTML
  • Create File Upload and Storage for book cover images
  • Setup Heroku and Connect to DB
  • Fine Fonts and Layout
  • Add a Rateing System for books

What roadblocks did you encounter and what did you do to overcome them?

There wasn't a lot of problems that I ran into just time management since im starting a second job soon and working on other finals so I wont be able to focus on this project as much.

However, I feel good about this project since I did put more hours and time into this week to try to finish most of the hard and heavy parts so other weeks can be easy and not take up so much time.

The only real hard part of this week was learning ESJ. I knew about EJS but never really had hands on experience with it, I usually just use handlebars however wanted to see how it was and like how its a lot more simple depending on how you need to use it.

It works for this project since its a lot of small views and just deplaying data however handlebars I like better if the views become more complicated and complex.
