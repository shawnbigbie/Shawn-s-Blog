Final Project - Update 3

Posted by Shawn Bigbie on Nov 20th, 2020

What have you done this week?

During this week (Nov 15th - Nov 21th) I had to restart my final project for DGM 3760 and decied to follow the default Recipe Book and Shopping List idea that the professor thought off. This New project is useing Express, Node Js, MongoDB for the backend, and Handlebars for the template engine and views for the application. The point of this project is for users to store and create, edit, and delete Recipes and to Create a Shopping List for ingredients while looking at Recipes from a public database.

Below are the things I have created and worked on.

  • Setup App.js and Dependencies
  • Setup Public Folder and StyleSheet
  • Create database on MongoDB for Recipe Book
  • Hooked up MongoDB to app.js
  • Setup Github Repo
  • Setup .env
  • Create recipe and ingredient model
  • Create recipe and ingredient routes
  • Create Recipe's Show, Add, Edit, and Index views and Routes
  • Create Home page, Shopping List Page and Partials
  • Create Project README

What do you plan on doing next week?

Things that I still need to work on or finish.

  • Create Shopping List Model
  • Create FrontEnd Style & View for Shopping List while viewing recipes
  • Create Shopping List's Show, Add, Edit, and Delete Routes
  • Setup Heroku and Connect to DB
  • Setup Cross Items off of a shopping list
  • Change quantities on a shopping list
  • Add a categorize filter system for recipes

What roadblocks did you encounter and what did you do to overcome them?

There was a couple of roadblocks that I encounterd this week, like having to restart since the last one was not approved. There was also some coding roadblocks that I was stuck on for a while like displaying all the recipes from the database and couldn't get an textarea field to display the stored data when you edit the recipe. I found some help on stack overflow for the displaying data from DB and then the textarea I switched to a input and it works now.

Besides that the rest of the time went smooth, I feel good about this project since I did finish most of the features. Finished the Recipe part of the application, now just have to create and integrate the Shopping List part of the assinment to the project.
